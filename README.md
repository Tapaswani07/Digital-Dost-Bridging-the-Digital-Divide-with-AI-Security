Digital Dost is an AI-powered safety ecosystem designed to protect the elderly from digital scams and social engineering. By utilizing Google’s Gemini AI and a vernacular-first approach, we empower seniors to navigate the digital world with dignity and security.

-- Overview
Elderly users are often the primary targets for cyber-crimes. Digital Dost acts as a "Digital Companion" that:

Intercepts suspicious links and messages in real-time.

Translates complex technical threats into simple, vernacular voice alerts (Marathi, Odia, Gujarati, etc.).

Synchronizes with a "Guardian Hub" app to keep family members informed and in control during emergencies.

-- Tech Stack (The Google Ecosystem)
This prototype is built using a modern, scalable web stack:

Frontend: HTML5, CSS3 (Material Design 3 principles), and Vanilla JavaScript.

Intelligence: Gemini 1.5 Flash API (Simulated/Integrated) for NLU/NLG scam analysis.

Backend/Database: Firebase (Authentication & Cloud Firestore for real-time sync).

Maps: Google Maps Platform for live SOS tracking.

-- Architecture
The solution is built on a 3-layer architecture:

User Interface Layer: High-contrast, accessible HTML interface designed for senior ergonomics.

Intelligence Layer: Gemini-powered analysis engine that processes incoming data for threats.

Infrastructure Layer: Firebase-powered real-time synchronization between the Senior and Guardian devices.

--Key Features
Zero-Click Interception: Automated scanning of suspicious SMS/Web activity.

Vernacular Voice Alerts: Reassuring AI personas that speak the user's native language.

One-Tap SOS: Immediate GPS broadcasting to family members via the Guardian Hub.

Remote Lockdown: Guardians can remotely secure sensitive banking apps if a threat is detected.
